**Call Center Offer Management Backend**

Welcome to the Call Center Offer Management Backend. This project provides an intuitive and efficient solution for managing customer offers in a call center environment. The backend allows call center agents to upload and manage customer offers with ease, ensuring a streamlined process for storing and retrieving offer information.

**Features**

Secure File Upload: Upload multiple customer offers simultaneously with file overwrite options.
User Management: Capture the uploader's name and timestamp for tracking purposes.
Local Storage: Utilizes browser local storage to keep data accessible even after page reloads.
File Management: Easily delete or update existing files with a single click.
Frontend Compatibility: Ready to integrate with a frontend interface for seamless customer interaction.

**Technologies Used**

HTML5 & CSS3 for the user interface.
JavaScript for handling file upload, data storage, and UI interactions.
Local Storage for storing and managing uploaded files.
GitHub for version control and project management.

**Usage**
Upload Offers: Select one or multiple files to upload. Provide your name when prompted, and the system will record the uploader's name along with the timestamp.
File Management: View all uploaded files in a neat list. You can delete any file if needed.
Local Storage: Your uploads are stored locally in your browser, ensuring persistence even after a page reload.

**Future Improvements**
Database Integration: Transition from local storage to a database (e.g., MongoDB, Firebase) for better scalability.
Advanced Search: Implement search functionality to easily filter offers by customer number, date, or uploader.
Authentication: Add user authentication for enhanced security and role-based access.
Frontend Integration: Connect to a dedicated frontend interface where agents can search for customer offers based on numbers and more.
